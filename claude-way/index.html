You've built your three skill files. Now you have a choice in how you go deeper — the self-guided HTML tools, or letting Claude do the analytical heavy lifting using your files as input.

Track 1 — HTML Tools

Work through the checklist and decision tree yourself. Good for understanding the framework deeply. Takes more time and attention.

AI-assisted
Claude does the analysis
Paste your files and a prompt into Claude. It reads across all three layers and returns structured findings you act on. Faster and often sharper.

The Claude Way Prompts

Three prompts — one for each analysis task. Copy the prompt, paste it into Claude along with your skill files, and Claude does the work.

Prompt 01Coherence Check
→
Paste your three skill files. Claude reads across all of them simultaneously and returns a structured report — what aligns, what creates tension, what contradicts, and what's missing. Equivalent to Layer 4.
Prompt 02Workflow & Routing Builder
→
Answer a few questions about how you work with AI, then paste your three files. Claude generates a system.md routing file and file annotations custom to your patterns. Equivalent to Layer 5.
Prompt 03Update Checker
→
Write what you want to add, select which file it's going into, paste your existing files. Claude reads the proposed addition against everything you've built and returns a clear verdict with a concrete next step.
How to use these prompts
01Open the prompt page — read the instructions and copy the full prompt text
02Open Claude.ai (or any Claude interface) and start a new conversation
03Paste the prompt, then paste your skill file contents where indicated
04Send. Claude returns the analysis — act on what it tells you
05Update your files, save them, and you're done
